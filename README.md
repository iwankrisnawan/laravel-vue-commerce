# laravel with vue
personal project, created by laravel 5 with vue

# setup backend
1. go to the backend folder
2. composer install
3. create database
4. Change `.env` file
5. `php artisan migrate:refresh --seed`
6. run `php artisan serve`

# setup frontend
1. go to the vueshop-ui folder
2. npm install
3. npm run serve
