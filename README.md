# laravel with vue
personal project, created by Laravel 5 & Vuetify

# setup backend
1. go to the backend folder
2. composer install
3. create database
4. Change `.env` file
5. `php artisan migrate:refresh --seed`
6. run `php artisan serve`

# setup frontend
1. go to the vueshop-ui folder
2. change `.env` if needed
3. npm install
4. npm run serve
